# Odin Sign-Up Form

This project is a sign-up form built as part of The Odin Project curriculum.

It includes a full-height background image sidebar, a styled logo section, and a registration form with custom input states using CSS. The layout was built with HTML and Flexbox.

Features:

- Background image with overlay
- Styled form inputs (:focus and :invalid states)
- Custom button styling
- External font for logo

Note: This project is not responsive and does not include JavaScript password matching validation.
